# Даалгавар 1: SpaceShip (Сансрын хөлөг)

**Оноо:** 25 | **Түвшин:** Дунд

## Зорилго

Сансрын хөлгийн үндсэн системийг загварчилсан `SpaceShip` класс бичнэ үү. Хөлөг нь зорчигч тээвэрлэх, нислэг хийх, түлш цэнэглэх боломжтой.

## Талбарууд (Fields) — бүгд `private`

| Талбар | Төрөл | Анхны утга | Тайлбар |
|--------|--------|------------|---------|
| `ner` | `String` | constructor-оос | Хөлгийн нэр |
| `tulsh` | `int` | `100` | Түлшний хувь (0-100) |
| `zorchigch` | `int` | `0` | Зорчигчдын тоо |
| `km` | `double` | `0` | Нийт нисэн зай |

## Constructor

```java
public SpaceShip(String ner)
```

- `ner` талбарт утга олгоно
- Бусад талбарууд анхны утгаараа эхэлнэ

## Methods

### 1. `zorchigchAvah(int too)` → `String`

Хөлөгт зорчигч нэмнэ.

- Хэрэв нэмсний дараа нийт зорчигч **10-аас их** болвол → `"Багтаамж хэтэрлээ!"` буцааж, зорчигч **нэмэхгүй**
- Амжилттай бол зорчигч нэмж → `"X зорчигч нэмэгдлээ"` буцаана (X = too)

**Жишээ:**
```java
SpaceShip ship = new SpaceShip("Falcon");
ship.zorchigchAvah(5);  // "5 зорчигч нэмэгдлээ"
ship.zorchigchAvah(6);  // "Багтаамж хэтэрлээ!" (5+6=11 > 10)
ship.zorchigchAvah(5);  // "5 зорчигч нэмэгдлээ" (5+5=10, яг 10 болохыг зөвшөөрнө)
```

### 2. `nisleg(double zai)` → `String`

Хөлөг нислэг хийнэ.

- Түлшний зарцуулалт: `zai * 0.5` (1 км-д 0.5 түлш)
- Хэрэв одоогийн түлш < шаардлагатай түлш бол → `"Түлш дутуу!"` буцааж, нислэг **хийхгүй**
- Амжилттай бол түлш хасаж, km нэмж → `"Нислэг амжилттай! X км нисэв"` буцаана (X = zai, double хэвээр)

**Жишээ:**
```java
SpaceShip ship = new SpaceShip("Falcon"); // tulsh=100
ship.nisleg(100);  // "Нислэг амжилттай! 100.0 км нисэв" (түлш: 100-50=50)
ship.nisleg(200);  // "Түлш дутуу!" (200*0.5=100 > 50)
```

### 3. `tulshTsenegleh(int hemjee)` → `void`

Түлш цэнэглэнэ.

- `tulsh += hemjee` гэхдээ **100-аас хэтрэхгүй**
- `Math.min(100, tulsh + hemjee)` ашиглана

**Жишээ:**
```java
ship.tulshTsenegleh(30);  // tulsh 50 → 80
ship.tulshTsenegleh(50);  // tulsh 80 → 100 (130 биш!)
```

### 4. `toString()` → `String`

**Формат:**
```
🚀 [нэр] | Түлш: X% | Зорчигч: Y | Нийт: Z км
```

**Жишээ:**
```
🚀 Falcon | Түлш: 100% | Зорчигч: 0 | Нийт: 0.0 км
```
